ВВЕДЕНИЕ
	Масштабирование изображения - изменение размеров (ширины и высоты) растра изображения с сохранением всего основного сюжета.
	Типы масштабирования:
1. Уменьшающее - размеры растра результирующего изображения меньше, чем размеры исходного;
2. Увеличивающее - размеры растра результирующего изображения больше, чем размеры исходного;
3. Смешанное - один из размеров растра результирующего изображения больше или равен соответствующему размеру исходного, а второй размер растра результирующего изображения, наоборот, меньше либо равен соответствующему размеру исходного.
	В данной работе будут рассмотрены простые алгоритмы уменьшающего масштабирования.
	При уменьшающем масштабирования изображение теряет информацию об объектах и становится менее чётким.
	Простейшими алгоритмами уменьшающего масштабирования являются следующие алгоритмы:
1. Прореживание [в n раз] - при масштабировании исходное изображение разбивается на фрагменты (тайлы) [n x n] пикселей и в результирующее изображение сохраняется яркость пикселя, расположенного ближе всего к середине соответствующего тайла;
2. Усреднение [в n раз] - при масштабировании исходное изображение разбивается на тайлы [n x n] пикселей и в результирующее изображение в качестве яркостей сохраняется среднее-арифметическое яркостей пикселей тайла.
	
ЗАДАНИЕ
	Написать консольное приложение на языке программирования C++ (или модифицировать любое из прерыдущих приложений), выполняющее следующие задачи:
1. Открытие изображения BMP (изображения в оттенках серого и цветные изображения); имя файла .bmp задаётся пользователем при запуске программы;
2. Масштабирование изображения описанными двумя способами - прореживанием и усреднением; уровень масштабирования n задаётся пользователем при запуске программы;
3. Замеры времени масштабирования для каждого способа в отдельности и вывод времени в консоль (в секундах с точностью 3 знака после запятой);
4. Сохранение масштабированного изображения в формате BMP; имена результирующих изображений задаются пользователем при запуске программы.

РЕЗУЛЬТАТ ВЫПОЛНЕНИЯ ЗАДАНИЯ
	Задание считается выполненным, если программа выполняет поставленные перед ней задачи - выполняет масштабирование изображения, сохраняет масштабированные изображения и выводит время масштабирования.
