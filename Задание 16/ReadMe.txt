ВВЕДЕНИЕ
	Особенностью срединного фильтра, реализованного в одной из прошлых работ, является то, что при выполнении свёртки все пиксели, участвующие в свертке, одинаково влияют на яркость результирующего пикселя, т.е. их влияние не зависит от дальности до центрального пикселя свёртки.
	Однако, в общем случае, для изображений это допущение не является верным.
	Существуют сглаживающие фильтры, выполняющие значения элементов матрицы которых принимают меньшие значения при отдалении от центра фильтра. Одним из таких фильтром является фильтр Гаусса.
	
	Фильтр Гаусса - сглаживающий свёрточный фильтр, матрица свёртки которого состоит из элементов, вычисленных с помощью двумерной функции Гаусса:
	
	A(i,j) = 1 / [2 * pi * S^2 * e^(((i-k/2)^2 + (j-k/2)^2)/(2*S^2))], где
	k - размер фильтра,
	S - СКО нормального распределения - параметр функции Гаусса, определяющий степень размытия изображения.
	Чем больше S, тем сильнее дальние пиксели влияют на результат свёртки (значения фильтра Гаусса стремятся к равномерному распределения), тем сильнее размывается изображение.
	
	При расчёте матрицы свёртки для различных k и S видно, что элементы матрицы, находящиеся на расстоянии 3*S от центра матрицы, по модулю становятся меньше ~0,001, что является очень маленькими коэффициентами, которые будут очень мало влиять на результат свёртки. Поэтому, на практике, S подбирается исходя из степени размытия которую необходимо достичь, а k выбирается не превосходящим 3*S.
	
	Для фильтра Гаусса, также, как и для остальных свёрточных фильтров, необходимо нормировать коэффициенты для сохранения средней яркости изображения.
		
ЗАДАНИЕ
	Написать консольное приложение на языке программирования C++, выполняющее следующие задачи:
1. Формирование матрицы свёртки фильтра Гаусса - размер фильтра k и СКО S задаются пользователем при запуске программы;
2. Нормализация матрицы свёртки фильтра Гаусса - деление элементов матрицы на сумму элементов этой же матрицы;
3. Если k < 15, то вывод в консоль обоих матриц свёртки в виде двумерного массива чисел, а также суммы элементов ненормализованной матрицы;
4. Сохранение нормализованной матрицы свёртки Гаусса в конфигурационный файл, используемый в предыдущем задании для выполнения свёртки (M = 1, O = 0); имя конфигурационного файла должно содержать k и S.
